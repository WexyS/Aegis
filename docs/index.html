Aegis

Aegis is a local-first, free-first Mission Control system for Windows. It is built for deterministic runtime truth, approval-aware automation, evidence-backed execution, replayability, and auditable local AI workflows.

Runtime health May remain fail

Runtime health and historical evidence/replay debt are not hidden or greenwashed. Unknown-era issues remain visible until explicitly handled by later gated work.

Evidence-backed execution

Aegis separates command dispatch from verification. Evidence exists does not mean verified, and missing or failed evidence cannot become success.

Policy and approval gates

Future capabilities must pass backend policy, risk, approval, and evidence expectations. Context, memory, model output, plugins, and frontend state are not permission.

Auditable local runtime

The project prioritizes replayable journaled behavior, backend-owned truth surfaces, maintenance diagnostics, and carefully gated capability expansion.
